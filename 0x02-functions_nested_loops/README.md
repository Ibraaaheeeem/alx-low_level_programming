## ALX Low Level Programming in C

#### ```functions and nested loops``` 
#### The following is List of scripts and their functions:

* 0-putchar.c: This program prints _putchar followed by a newline

* 1-alphabet.c: This program prints the alphabets in lower case

* 2-print_alphabet_x10.c: This program prints lowercase alphabets followed by a newline 10 times

* 3-islower.c: This program checks whether a character is a lowercase or not

* 4-isalpha.c: This program checks whether a charcater is an alphabet or not

* 5-sign.c: This program prints the sign of an input number + is positive, 0 if zero and - if negative

* 6-abs.c: This program prints the absolute value of an integer

* 7-print_last_digit.c: This program prints the last digit of a number

* 8-24_hours.c: This program prints every minute of the day of Jack Bauer, starting from 00:00 to 23:59

* 9-times_table.c: This program prints the 9 times table starting with 0

* 10-add.c: This program reads two integers and return the result

* 11-print_to_98.c: This program prints a set of all natural nnumbers from n to 98

* 100-times_table.c: This program prints n times table starting with 0
